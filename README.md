# Predictive-maintenance-system
AI-powered predictive maintenance system that monitors temperature and vibration in real time using embedded sensors and edge devices. It analyzes patterns to detect anomalies early, enabling proactive fault detection, reduced downtime, and improved system reliability.This project implements a real-time IoT monitoring system using QNX RTOS on Raspberry Pi. It captures sensor data, processes it with deterministic behavior, and performs predictive analysis for anomaly detection. The system ensures reliability, efficient resource utilization, and real-time alerts for industrial applications.
